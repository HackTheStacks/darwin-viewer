# Darwin Viewer

> View Darwin's notes! Match Darwin's notes! Basically - be Darwin.

## Install

```bash
git clone https://github.com/HackTheStacks/darwin-viewer
cd darwin-viewer
npm install
```

## Usage

```bash
npm start
```

## API

### `GET /some-endpoint/:a-param/view?an-option="a-value"`

**Params:**
- `a-param`: Some param.

**Options:**
- `an-option`: Some option.

**Response:**
```json
{
	"id": 100
}
```
